# Testimonial Cards

A modern and visually appealing testimonial cards component showcasing customer feedback. This project features multiple card styles with a minimalist design using HTML and CSS.

## Features
- Multiple testimonial card layouts
- Clean and minimalist color theme
- Image carousel with navigation
- Responsive design
- Smooth animations and transitions
- Easy to customize and integrate

## Demo
[View the Testimonial Cards Project Page](https://roadmap.sh/projects/testimonial-cards)

## Getting Started

### Prerequisites
- A modern web browser
- No build tools or frameworks required

### How to Run the Project
1. Clone or download this repository.
2. Navigate to the `frontend/5-testimonial-cards` directory.
3. Open the `index.html` file in your web browser.

```
frontend/
└── 5-testimonial-cards/
    ├── index.html
    ├── styles.css
    └── README.md
```

### Customization
- Edit the `index.html` file to update testimonial content, author names, and descriptions.
- Modify `styles.css` to change colors, fonts, and styling as needed.
- Replace `unknown-pic.jpg` with your own author images.

### Project Structure
- **Card Type 1**: Chat bubble style testimonial with author profile
- **Card Type 2**: Image and content side-by-side layout with star rating
- **Card Type 3**: Image carousel with navigation arrows
