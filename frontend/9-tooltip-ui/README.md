# Tooltip UI

A simple tooltip UI component built with HTML and CSS. Hover over the navigation links to see tooltips appear with smooth fade-in animations.

**Project Page:** https://roadmap.sh/projects/tooltip-ui

## How to Run

1. Open `index.html` directly in your browser, OR
2. Use a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (with http-server installed)
npx http-server

# Using PHP
php -S localhost:8000
```

Then visit `http://localhost:8000` in your browser.

## Features

- CSS-only tooltips using pseudo-elements (`::before` and `::after`)
- Smooth fade-in animation on hover
- Tooltip arrow indicator
- Centered tooltip positioning below navigation items
