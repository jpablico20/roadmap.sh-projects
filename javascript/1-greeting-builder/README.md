# JS Greeting Builder

Project: https://roadmap.sh/projects/js-greeting-builder

## Run

This project is a simple JavaScript exercise.

### Requirements
- Node.js (any modern version)

### Steps
1. Open a terminal in the `javascript/` folder.
2. Run:

```bash
node greetingBuilder.js
```

## Expected Output

```txt
Good morning, Jorim Pablico!
Good afternoon, Jorim Pablico!
```
